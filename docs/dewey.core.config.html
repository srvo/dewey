Manages configuration settings for the application.

This class inherits from BaseScript and provides methods for loading and accessing configuration values.

Initializes the ConfigManager.

Parameters:

  • config_section (str) – The section in the configuration file to use.

  • db_connection (DatabaseInterface | None) – An optional database connection to use. Defaults to None, which will create a connection.

  • motherduck_connection (MotherDuckInterface | None) – An optional MotherDuck connection to use. Defaults to None, which will create a connection.
