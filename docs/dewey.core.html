dewey.core.base_script module¶

Base class for all Dewey scripts.

This module implements the BaseScript class, which serves as the foundation for all scripts in the Dewey project. It provides standardized access to: - Configuration management (via dewey.yaml) - Logging facilities - Database connections - LLM integrations - Error handling

All non-test scripts MUST inherit from this class as specified in the project conventions.
