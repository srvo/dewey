Base class for automation modules within Dewey.

This class provides a standardized structure for automation scripts, including configuration loading, logging, and a run method to execute the script’s primary logic.

  • db_conn (DatabaseConnectionInterface | None) – Optional database connection. Defaults to None, which will use the BaseScript’s default.
