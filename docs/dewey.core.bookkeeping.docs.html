class dewey.core.bookkeeping.docs.DocsModule(name, description='Documentation Module', documentation_task=None)[source]¶

Bases: BaseScript

A module for managing documentation tasks within Dewey.

This module inherits from BaseScript and provides a standardized structure for documentation-related scripts, including configuration loading, logging, and a run method to execute the script’s primary logic.

Initializes the DocsModule.

Parameters:

  • name (str) – The name of the module.

  • description (str, optional) – A brief description of the module. Defaults to “Documentation Module”.

  • documentation_task (DocumentationTask, optional) – The documentation task to execute. Defaults to None.
