execute()[source]¶

Execute the script.

This method should be implemented by all subclasses to define the main functionality of the script. It should handle:

  1. Setting up any required resources (DB, LLM, etc.)

  2. Running the script’s main functionality

  3. Cleaning up resources

  4. Handling exceptions
